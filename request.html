Creating a Document Request

This section will give you step-by-step guidance on creating a Document Request in SIAConnect.

Document Request

  • To start a document request, simply type "Document Request" in the SIAConnect chat window.
  • SIAConnect will respond with a document request form.
  • Select which document type you are wanting (e.g., Tech bulletin, User Manual, Knowledge Base article).
  • Select the product model that the request pertains to. If the model is not in the list, you can manually type in the model.
  • Add a description - this is everything that you want to make sure is completed and included in the document. The more information, the better!
  • Set a priority. If there is no rush (can wait a week or longer), set the priority to Low. If there's not a huge rush but you need it within 5-7 days, select Medium. For urgent requests, set the priority as High.
  • Once you have filled out the form, select OK.
  • SIAConnect will then show you the details of your request for you to confirm that everything is correct.
  • If everything looks good, select Submit Request. If you notice an error or wish to add more information, select Edit Request to make changes.
  • Once you have submitted your request, SIAConnect will respond with a document request number.
  • You will also receive an email confirmation of your request, which includes the document request number.
